# Channel Specifications

## Telegram (Primary — 80% effort)

| Parameter | Value |
|-----------|-------|
| Language | Russian |
| Length | 800-2000 characters |
| Frequency | 3 posts/week |
| Timing | Weekdays 9-11 or 19-21 Moscow time |
| Audience | CIS founders, marketers, no-coders |

**Format specifics:**
- Use **bold** for emphasis (sparingly)
- Line breaks for readability
- Emoji: minimal to none (not infobiz style)
- Links: sparingly, prefer DM CTA
- Visuals: architecture diagrams, screenshots when relevant

**Content mix:**
- Case breakdowns: 1/week
- Micro-insights: 2/week
- Build-in-public updates: 1/week
- Polls/engagement: 1-2/month

**CTA style:** Soft, embedded in value. "Write 'audit' in DM" not "BUY NOW!!!"

---

## Twitter/X (Secondary — 15% effort)

| Parameter | Value |
|-----------|-------|
| Language | English |
| Format | Threads (5-15 tweets) or single tweets |
| Frequency | Adapt best Telegram content |
| Audience | International AI/tech community |

**Format specifics:**
- Threads: numbered, clear progression
- Single tweets: punchy insights, under 280 chars
- Engage with AI community (comments, quotes)
- Build connections with Western AI builders

**Content approach:**
- Adapt best-performing Telegram content
- More technical depth acceptable
- Build-in-public resonates well
- Case studies work as threads

---

## Cross-posting Notes

- Don't translate literally — adapt for context
- Telegram audience more skeptical of hype
- Twitter audience expects more directness
- Both hate infobiz tone equally
